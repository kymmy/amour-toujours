# Le Village — Amour Toujours

**Initiative Solidaire Locale** — Encouraging mutual aid between residents in Marseille 11e & 12e.

**Live:** https://amour-toujours.le-village-marseille.fr

**Portal:** https://le-village-marseille.fr

---

## What is it?

Amour Toujours is a local solidarity initiative that brings neighbours together through shared actions: solidarity dinners, pink collection bins in partner shops, entrepreneur meetups, and charity wine tastings. All proceeds go to supporting families in need.

Includes a dedicated event page for the **Cafe des Entrepreneurs Solidaires** — a monthly morning meetup for expats, entrepreneurs, and remote workers.

## Pages

- `/` — Main dossier: vision, actions, partners, transparency
- `/cafe-entrepreneurs` — Monthly entrepreneur meetup event page

## Tech Stack

- Next.js 16 (App Router)
- TypeScript (strict)
- Tailwind CSS 4
- Framer Motion
- Bilingual FR/EN

## Development

```bash
npm install
npm run dev    # http://localhost:3000
```
